# 통화정책 전달에 따른 주택담보대출 금리 변화가 서울 아파트 거래량에 미치는 영향

## 프로젝트 개요
본 프로젝트는 한국은행 기준금리 변화가 주택담보대출 금리를 거쳐 서울 아파트 거래량에 미치는 영향을 분석하는 것을 목표로 한다.

기준금리 → 주택담보대출 금리 → 거래량의 전달 구조를 단계적으로 확인하고,
추가적으로 소비심리와 보유세 부담 변수를 포함하여 설명력을 확장할 계획이다.

---

## 연구 질문
1. 기준금리 변화는 주택담보대출 금리에 유의미한 영향을 미치는가  
2. 주택담보대출 금리 변화는 서울 아파트 거래량 변화에 영향을 미치는가  
3. 소비심리와 보유세 부담은 거래량 설명에 추가적인 영향을 가지는가  

---

## 사용 데이터

| 변수명 | 설명 | 출처 |
|--------|------|------|
| base_rate | 기준금리 | 한국은행 ECOS |
| mortgage_rate | 주택담보대출 금리 | 한국은행 ECOS |
| trade_volume | 서울 아파트 거래량 | 국토교통부 실거래가 공개 시스템 |

분석 기간: 2020.01 ~ 2024.12
